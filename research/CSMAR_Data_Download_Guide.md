## 二、各模块详细下载步骤

### 模块1：公司基本信息
- **路径**: 股票市场 → 公司研究系列 → 公司基本信息
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 股票简称 (Stknme)
  - 上市日期 (Listdate)
  - 注册省份/办公省份 (Province)
  - 行业代码 (Industry_code, 证监会2012版)
- **时间**: 全部（用于筛选2013年前已上市的公司）
- **导出格式**: Excel

### 模块2：年度财务数据
- **路径**: 股票市场 → 财务报表 → 财务指标分析
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度 (Accper / Year)
  - 总资产对数 (A001000 → 取log)
  - 资产负债率 (F091301A)
  - 总资产收益率ROA (F050301A)
  - 营业收入 (F010101A)
  - 营业收入增长率 (计算: 本年/上年 - 1)
- **时间**: 2013-2023
- **导出格式**: Excel

### 模块3：无形资产明细（关键！）
- **路径**: 股票市场 → 财务报表 → 无形资产明细表
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 无形资产原值总额
  - **土地使用权**
  - **软件/系统/信息系统**（关键！这是 IT_Invest 的代理变量）
  - **其他无形资产**
- **时间**: 2013-2023
- **导出格式**: Excel
- **备注**: 如果"软件/系统"单独列示不清晰，可用"无形资产总额 - 土地使用权"作为近似

### 模块4：公司治理数据
- **路径**: 股票市场 → 公司治理 → 董事会治理
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 董事会人数 (Bddsize)
  - 独立董事人数 (Inddnum)
- **时间**: 2013-2023

### 模块5：股权性质
- **路径**: 股票市场 → 公司治理 → 股权性质
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 第一大股东持股比例 (Shrcrf1)
  - 实际控制人性质 (State-owned vs Private)
- **时间**: 2013-2023

### 模块6：数字化转型指数
- **路径**: 创新经营系列 → 数字经济/数字化转型/企业数字化转型
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 数字化转型总指数
  - 底层技术（云计算、大数据、AI、区块链等分项指数）
- **时间**: 2013-2023
- **备注**: 如果CSMAR没有单独的子库，可以用年报文本挖掘替代：
  - 搜索年报中包含"ERP"/"企业资源计划"/"信息化"/"数字化"/"SAP"的关键词

### 模块7：政府补贴数据
- **路径**: 股票市场 → 财务报表 → 政府补助明细
- **选择**: 全部A股
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 政府补助总额
  - 与信息化/数字化相关的补助（如有分类）
- **时间**: 2013-2023

### 模块8：行业分类
- **路径**: 股票市场 → 公司研究 → 行业分类
- **字段**:
  - 股票代码 (Stkcd)
  - 会计年度
  - 证监会行业分类代码（2012版）
  - 是否为高科技行业
- **时间**: 2013-2023

## 四、样本筛选规则

1. 保留 2013-2023 年 A 股上市公司
2. 剔除金融行业（证监会行业代码 J）
3. 剔除 ST、*ST 公司
4. 剔除关键变量缺失的观测
5. 剔除 2013 年之后新上市的公司（确保有足够上市历史）
6. 连续变量上下 1% 缩尾（winsorize）

---

## 五、下载注意事项

1. **所有数据都选择"面板数据"格式**（同一公司多年数据）
2. **股票代码格式统一**：用6位代码，不要前缀
3. **年度选择"会计年度"**而非"公告年度"
4. 导出时选择"Excel格式"，不要CSV（保留字段名）
5. 每个模块下载后重命名：
   - `module1_company_info.xlsx`
   - `module2_financial.xlsx`
   - `module3_intangible.xlsx`
   - `module4_governance.xlsx`
   - `module5_ownership.xlsx`
   - `module6_digital.xlsx`
   - `module7_subsidy.xlsx`
   - `module8_industry.xlsx`
   - `policy_timeline.xlsx`（手工整理）
